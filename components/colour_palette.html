Colour Palette

Colour palette is what you need for zanui colours.

Basic

You can change background by adding css class like this.

HTML

<div class="bg-blue"></div>
<div class="bg-light-grey"></div>
<div class="bg-green"></div>
<div class="bg-yellow"></div>
<div class="bg-light-blue"></div>
<div class="bg-red"></div>

SCSS

You can refer colour value by using variables.

// Black and White
$color-black: #000000;
$color-white: #FFFFFF;

// Basic
$color-blue: #385673;
$color-light-grey: #E7E7E7;
$color-green: #4E9458;
$color-yellow: #FDBC51;
$color-light-blue: #4D8FA9;
$color-red: #E9413E;

// Typography
$color-blue-smoke: #7993A3;
$color-slate: #667079;

// Fancy
$color-forest: #5A9463;
$color-salmon: #D45257;
$color-teal: #5AA6A2;
$color-tangerine: #FD9251;
$color-persimmon: #E95D3E;
$color-seafoam: #D3EEE2;
$color-thistle: #8C84B0;
$color-pink: #E593B8;
