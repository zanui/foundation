Colourful Text

Colourful text is what you need for zanui colorful design.

Text List

You can change text color by adding css class like this.

HTML

<p class="txt-blue">...</p>
<p class="txt-light-grey">...</p>
<p class="txt-green">...</p>
<p class="txt-yellow">...</p>
<p class="txt-light-blue">...</p>
