You can create a icon using minimal markup.

HTML

Prefix & Postfix Icons

You can create a prefix or postfix icon using minimal markup.

HTML

<span class="z-icon-tree prefix-icon"></span>
<span class="z-icon-tree postfix-icon"></span>
Do not include extra space for them in HTML

Spinning Icons

You can create a spinning icon using minimal markup.

HTML

<span class="z-icon-star spinning"></span>
<span class="z-icon-returns spinning"></span>
<span class="z-icon-cross-cancel spinning"></span>
<span class="z-icon-help spinning"></span>

Rotated & Flipped Icons

You can create a rotated and flipped icon using minimal markup.

HTML

<span class="z-icon-edit"></span>
<span class="z-icon-edit rotate-90"></span>
<span class="z-icon-edit rotate-180"></span>
<span class="z-icon-edit rotate-270"></span>
<span class="z-icon-edit flip-horizontal"></span>
<span class="z-icon-edit flip-vertical"></span>
